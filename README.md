# PARKING-SYSTEM
Sistem Parkir Otomatis (Project PWL)
Deskripsi
Sistem Parkir Otomatis adalah aplikasi berbasis web yang dibuat untuk membantu pengelolaan parkir secara efisien dan terstruktur. Sistem ini memungkinkan pencatatan kendaraan masuk dan keluar, menghitung biaya parkir secara otomatis, serta memonitor kapasitas parkir yang tersedia.

Project ini dibuat sebagai tugas Pemrograman Web Lanjut (PWL).
Tujuan
- Mempermudah proses pencatatan kendaraan masuk & keluar
- Mengurangi kesalahan perhitungan biaya parkir
- Menyediakan informasi slot parkir secara real-time
- Melatih implementasi CRUD dan database dalam web
Fitur Utama
- Input kendaraan masuk (plat nomor, jenis kendaraan, waktu masuk)
- Pencatatan kendaraan keluar
- Perhitungan otomatis biaya parkir
- Monitoring jumlah slot parkir
- Riwayat data parkir
- Manajemen data kendaraan
